# clips/

Raw and lightly-processed gameplay video, vertical/portrait (9:16) to match the game's native orientation. Source material for the platform agents (`agents/tiktok-agent.md`, `agents/instagram-agent.md`, `agents/youtube-agent.md`) and the Script Writer Agent.

Naming convention: `{event_type}_{event_id}_{descriptor}.mp4` — see `content-assets/README.md` for the full convention and lifecycle. Each clip should have a paired sidecar in `../metadata/` with the same `{event_type}_{event_id}` prefix.

Populated manually until the auto-clipping system (`automation/auto-clipping.md`) is built, per the sequencing in `ROADMAP.md`.
