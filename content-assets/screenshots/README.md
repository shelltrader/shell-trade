# screenshots/

Still frames captured at significant in-game moments — boss defeats, rank-ups, achievements unlocked. Used for carousels, thumbnails, Stories, and any platform/pillar combination that doesn't need motion (see `marketing/content-pillars.md` and `marketing/social-strategy.md` for which formats call for stills vs. clips).

Naming convention: `{event_type}_{event_id}_{descriptor}.png` — see `content-assets/README.md`. Full resolution, native portrait aspect ratio; downsampling happens per-platform downstream, not at capture time.
