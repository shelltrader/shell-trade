# scripts/

Script Writer Agent output (`agents/script-writer.md`): master scripts and per-platform adaptation notes, kept as plain text/markdown so they're diffable and reviewable like any other content in this repo, distinct from rendered media in the other `content-assets/` subfolders.

Naming convention: `{event_type}_{event_id}_{descriptor}.md` — see `content-assets/README.md`. Each script should reference the raw footage/screenshot it was written against and carry forward the Trading Educator Agent's "teaching note" verbatim (per `agents/trading-educator.md`) so the substance never drifts during platform adaptation.
