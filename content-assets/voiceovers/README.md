# voiceovers/

Generated narration audio (ElevenLabs output, per `automation/architecture.md`), each file keyed to a specific script in `../scripts/`. Used when a platform agent's brief calls for narration rather than on-screen-text-only delivery (see `agents/tiktok-agent.md` and `agents/youtube-agent.md`).

Naming convention: `{event_type}_{event_id}_{descriptor}.mp3` (or the project's chosen audio container) — see `content-assets/README.md`. Should use a single consistent voice profile across all output so narration stays recognizable as "Shell's voice" when presented in-character, per `marketing/brand-voice.md`.
