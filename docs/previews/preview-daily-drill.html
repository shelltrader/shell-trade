A fresh challenge every day, with a streak that multiplies your Shell reward — the loop that brings players back daily (the lever for paid-ads retention).

What changed

● BEFORE

  • No reason to return on a given day.
  • Learning only happened mid-run.
  • No habit loop or streak.

● AFTER

  • Finishing builds a day streak; the streak multiplies Shells (×1 → ×3 by day 9).
  • Same daily for everyone (seeded by date), stable if reopened; locks until tomorrow once done.
  • Reuses the existing quiz pool (32 questions) — no new content needed.
